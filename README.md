# 💍 Invitación de Boda - Agos & Martin

¡Bienvenidos al repositorio de la invitación digital para la boda de Agos y Martin!

Este proyecto fue desarrollado como parte de mi formación en **Análisis y Desarrollo de Software**, aplicando conceptos de programación moderna y diseño web.

## 🚀 Tecnologías Utilizadas

- **Next.js 15** (App Router)
- **React** (Hooks y Componentes)
- **Tailwind CSS** (Diseño Responsivo)
- **TypeScript** (Tipado fuerte)
- **Git & GitHub** (Control de versiones)

## ✨ Características

- **Countdown**: Contador en tiempo real para el gran día.
- **Galería Interactiva**: Visualización de fotos con efecto modal.
- **Confirmación RSVP**: Formulario integrado para invitados.
- **Lluvia de Sobres**: Sección detallada con información de regalos.
- **Google Maps**: Ubicación del evento integrada.
